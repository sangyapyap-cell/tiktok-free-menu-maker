🎨 Lomu Menu

A tool to create menu images using TikTok gift images.

🚀 Features
🖼️ Add images (Local / URL / TikTok gifts)
✏️ Text editing (font, bold, italic)
📐 Auto alignment (horizontal / vertical / grid)
🧩 Layer system
📦 TikTok gift database update
💾 Export transparent PNG / optimized PNG
🧩 How to Use
Launch the program
Click Import Image or Search TikTok Gifts
Select the image or gift
Add and customize text
Use alignment tools
Export as PNG
🌐 Data Sources

This program collects gift images from public web pages:

https://streamtoearn.io/gifts
https://tiktokboxgames.com/gifts.htm
https://zn-tiktok.com/gift/index.php

※ Not an official TikTok API.
